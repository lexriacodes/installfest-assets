Frequently Asked Questions

Will I lose my Windows files? expand_more
No, as long as you follow our backup guide! We recommend doing a "Dual Boot" setup which allows you to keep Windows 11 and Linux on the same computer, choosing between them when you turn it on.
Is Linux hard to use for a beginner? expand_more
Modern Linux distributions like Linux Mint or Ubuntu are very similar to Windows. You’ll have a taskbar, a start menu, and all your favorite web browsers like Chrome and Firefox.
Is this really free? expand_more
Yes! Both the Linux operating system and the help provided by our volunteers are 100% free of charge. This is a community service event.
What version of Linux will I get? expand_more
We primarily recommend Linux Mint for Windows 11 switchers, but we have Ubuntu, Fedora, and Pop!_OS available based on your hardware and preference.
