# 🐧 Linux Install Fest – Community Event Assets

This repository contains all assets, copy, and materials for our **Linux Install Day**, a one-day public event focused on helping people explore and install Linux on their devices.

## 📍 Event Overview

We’re hosting a hands-on community event at our local library where attendees can:

* Install Linux on desktops, laptops, servers, or embedded systems
* Get help troubleshooting or fixing existing Linux installs
* Learn about open-source tools and workflows
* Explore Linux as a daily driver (yes—even for gaming 🎮)

Whether someone is brand new or already technical, this event is designed to meet them where they are.

## 🎯 Who This Is For

* Windows users frustrated with Copilot, bugs, or data loss
* Developers curious about Linux environments
* Students and hobbyists exploring tech
* Gamers interested in running Steam on Linux
* Anyone who wants more control over their system

## 🛠️ Contributing

We welcome contributions from organizers and collaborators.

**Before submitting changes:**

* Keep messaging consistent with the event tone (accessible, welcoming, non-elitist)
* Avoid distro wars—this is about helping people, not debating preferences
* Ensure anything public-facing is easy to understand for non-technical users

**How to contribute:**

1. Create a branch
2. Make your changes
3. Open a pull request with a clear description

## 🧠 Messaging Guidelines

* Focus on **freedom, control, and reliability**
* Avoid jargon unless clearly explained
* Be welcoming to beginners
* Highlight real benefits (performance, privacy, customization)
* Acknowledge common concerns (compatibility, learning curve)

## ⚠️ Important Notes for Attendees

(Used in flyers and guides)

* Back up your data before attending
* Bring your device and charger
* Bring any important login credentials (if needed)
* Installs may take time depending on hardware

## 🤝 Community Focus

This event is about **education, accessibility, and empowerment** through open-source software.
No pressure, no gatekeeping—just helping people learn and try something new.
